Movie Info API Documentaion

Zalmy's Movie Info Database and API

Zalmy created this application that enables you to store and manage all the things that you love about movies. You can create your database and filter it by name, genre, actors, year, and much more. This allows you to keep track of all your favorites like what you already watched, what you wanna watch, and much more!

Input fields to include in the development process

  • id
  • Name
  • favoriteMovies

Categories to include for movies

  • Title
  • Description
  • Genre: Name, Description
  • Director: Name, Bio, Birth
  • ImageUrl
Logic Endpoint URL Request type Query params Request format Response format
Return all movies /movies GET All movies n/a JSON
Return a JSON with movie title /movies/:Title GET Return a single movie by it title n/a JSON
Return JSON with genre info /movies/genre/:Name GET Get the genre info by using the genre name n/a JSON
Return JSON with director info /movies/Director/:Name GET Pull the director info using the director name n/a JSON
Add new user /users POST Add a new user using the models params JSON JSON
Update user /users/:Username PUT Update the required fields of a current user JSON JSON
Delete a movie from a user /users/:Username/movies/:MovieID DELETE Delete a movie from the favoriteMovies of a user n/a JSON
Add favoriteMovies to a user /users/:Username/movies/:MovieID POST Add a movie id to a users favoriteMovies n/a JSON
Delete a user /users/:Username DELETE Allow a user to deregister n/a 200
